setattr、getattr

(attr——attribute)

setattr用于设置对象的属性值，getattr用于获取对象的属性值。相比于 “.” 功能更丰富。

  • 当实例a有属性bar时，当使用setattr(a, ‘bar’, 5)时，会将实例a属性bar设置为5;
  • 当实例a无属性bar时，会为实例a创建一个实例属性bar，赋值为5，而类属性不变。
  • getattr当属性不存在时避免抛出AttributeError，若设置默认值，则不会报错。例如，getattr(a, ‘baz’, ‘default’)会返回’default’，而a.baz则会引发异常。

all 、any

  • all：是否都为True any：是否存在True

  • all(iterable, /)

    1. 判断给定的可迭代参数 iterable 中的所有元素是否都为 TRUE，如果是返回 True，否则返回 False。
    2. 元素除了是 0、空、None、False 外都算 True。
    3. 空元组、空列表返回值为True，这里要特别注意。
    4. 不会进入内层。

print(all(['a', 'b', '', 'd']))   # 存在一个为空的元素，False
print(all([[''],['']]))			 # 不会进入内层，首层元素['']不为0、空、None、False
  • any(iterable, /)
    • “是否存在 True”的意思，True代表的是 非空类数据。

sorted

  • sorted() 对 所有可迭代的对象 进行排序操作。

  • sort 与 sorted 区别：

    sort 是 list 的方法，sorted 可对所有可迭代的对象应用。

    list 的 sort 方法返回的是对已经存在的列表进行操作，而内建函数 sorted 方法返回的是一个新的 list，而不是在原来的基础上进行的操作。

  • sorted(iterable, key=None, reverse=False)，三个参数，依次是 可迭代变量、选择内层元素的函数方法、升序or降序。

class Circle:
    def __init__(self, radius):
        self._radius = radius  # 私有变量
        self._pi = 3.141592653589793
    
    @property
    # 通过方法返回变量，起到保护_radius作用（只读）
    def radius(self):
        return self._radius
    
    @property
    # 实时计算面积area
    def area(self):
        return self._pi*(self._radius**2)
    
    @radius.setter
    # 控制radius的修改
    def radius(self, value):
        if value < 0:
            raise ValueError("半径不能为负")
        self._radius = value  # 合法时存储
    
    @radius.deleter
    # 控制radius的删除
    def radius(self):
        print("禁止删除半径！")
        
c = Circle(5)
print(c.radius) # 5
print(c.area)	# 78.53981633974483

动态计算属性值 ❌ 需手动计算 ✅ 实时计算（如 面积 = πr²） 属性值随依赖变量自动更新，无需额外调用方法

    @classmethod
    def create_parser(cls, file_path):
